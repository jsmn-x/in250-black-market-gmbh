# Security Policy
 
## Supported Versions
Dieses Projekt wird aktiv gepflegt. Sicherheitsupdates werden über den `main` Branch veröffentlicht.
 
## Reporting a Vulnerability
 
Wenn du eine Sicherheitslücke findest, melde diese bitte verantwortungsvoll.
 
Bitte sende eine Nachricht mit folgenden Informationen:
 
- Beschreibung der Sicherheitslücke
- Schritte zur Reproduktion
- Betroffene Dateien oder Komponenten
- Mögliche Auswirkungen
 
Du kannst das Problem melden über:
 
- GitHub Issue (wenn es keine sensible Information enthält)
- Direkt an den Maintainer über GitHub
 
Bitte veröffentliche keine Sicherheitslücken öffentlich, bevor sie behoben wurden.
 
Vielen Dank für deine Unterstützung bei der Sicherheit des Projekts.
